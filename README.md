### Установка

Добавить `"./vendor/4geo35/variation-cart/src/resources/views/components/**/*.blade.php",
        "./vendor/4geo35/variation-cart/src/resources/views/web/**/*.blade.php",
        "./vendor/4geo35/variation-cart/src/resources/views/livewire/web/**/*.blade.php",` в `tailwind.config.js`, созданный в пакете `tailwindcss-theme`.


#### Commands

Очистить устаревшие корзины:

    php artisan clear:old-carts
